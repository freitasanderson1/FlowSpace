# Projeto de Rede Social com chat utilizando websockets.
## Tecnologias utilizadas:
#### Backend:
- Channels 4.1.0
- Channels-redis 4.2.0
- Django 5.1.2
- Django Channels Rest Framework 1.2.0
- Django Rest Framework 3.15.2
- MaterialDash 0.0.22
- Redis 5.1.1
- Uvicorn 0.32.0
- Websockets 13.1

#### Frontend:
- Next.js
- Tailwind

#### Mobile:
- React Native
- Expo
